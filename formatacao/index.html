Elementos de Formatação

No HTML podemos escrever utilizando negrito, itálico ou ambos para destacar partes de um texto.

Para isso, era utilizado as tags '< b > ,< /b>' e '< i>, < /i>' nas palavras selecionadas.

No HTML5 as tags utilizadas são o ' < strong>, < /strong>, para deixar algo em negrito e < em>, < /em>, para deixar algo em itálico.
