Estudo sobre Links

É possivel criar ancoras ou links dentro do conteudo exibido na página.
Para isso basta utilizar o símbolo '#' seguido de um id dentro do atributo href.
Como, por exemplo: '< a href="#titulo-1">Título 1< /a >'.
O id é o atributo que sera adicionado dentro da tag na qual você pretende transformar
em seção navegável. Por exemplo, '< h2 id="titulo-1">Título 1 /h2 >'.

Para navegar para links externos, deve-se utilizar todo o endereço da página mostrado no navegador
podendo ou não adicionar o www após o uso do protocolo https.
Para navegar para a página do Google, utilizamos a tag 'a', junto de seu atributo 'href'
e o endereço : https://www.google.com/.
Podemos adicionar o atributo 'target', mais sua propriedade "blank", para que o link clicado abra uma nova aba ou guia
no navegador, em vez de atualizar a pagina em que o usuário está. Desse modo, clicando em Google
uma outra aba é aberta.
